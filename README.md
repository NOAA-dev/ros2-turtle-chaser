# ROS 2 Turtle Chaser

A ROS 2 project demonstrating a multi-node system with a controller and manager node using turtlesim.

## Features
- Multi-node ROS 2 architecture
- Custom services and messages
- Async service handling
- Proportional control for smooth pursuit
- Dynamic spawn and kill logic

## Requirements
- ROS 2 Jazzy
- turtlesim
